C语言考试及学习

常用的功能函数

  1. 计算S=1- 1/2 + 1/3 +…-1/2*n

    double fun(int n)
{
int i;
double sum=0.0;
if (n>1&&n<=100){
  for(i=1; i<=n; i++)
    sum+=1.0/(2*i-1)-1.0/(2*i);      //sum =sum +(1/(2*i-1)-1/(2*i))
}
return sum;
}

  2. 将a、b中的两个两位正整数合并形成一个新的整数放在c中。

    合并的方式是：将a中的十位和个位数依次放在变量c的十位和千位上，b中的十位和个位数依次放在变量c的个位和百位上。

    void fun(int a, int b, long *c)                  //*c为c指针变量的内容，*为取内容运算符。
{
*c=(a%10)*1000+(b%10)*100+(a/10)*10+(b/10);      //取余，取整
}

  3. 求整数x的y次方的低3位值。例如，整数5的6次方为15625，此值的低3位值为625。

    long fun(int x,int y,long *p)    
{  int i;    
   long t=1;    
   for(i=1; i<=y; i++)    //用以用 pow(x,y)函数
      t=t*x;    
     *p=t;    
   t=t%1000;    
   return t;    
}

  4. 将tt所指字符串中的小写字母都改为对应的大写字母，其他字符不变。

    char* fun(char tt[])
{
  int i;
  for(i = 0; tt[i]; i++)
    if(('a'<=tt[i])&&(tt[i]<='z'))
      tt[i]-=32;          //小写转换成大写字母则ASCII减32
  return(tt);
}

  5. 保留一个实数的两位小数，从第三位实现四舍五入。例如输入12.3785，处理后输出12.38。需要注意的是不允许直接采用输出控制以”%m.nd”的形式来得到输出结果。

    double fun(double x)
    {
    x=(int)(x100+0.5); /移动小数点/
    x/=100; /    舍掉小数点后两位*/
    return x;
    }

  6. 输入一个字符型的数字，并转换成相应的整数输出。如读入字符’8’，转换成整数8输出。

    int fun(char ch) {
   int d;
   d=ch-'0';           //也可以用 d = ch-48;
   return d;
}

  7. 输入一个两位整数，将其个位和十位改换位置，然后输出。

    int fun(int m)
    {
    int n;
    n=m%10; /取余获得个位数/
    m=n10+m/10; /十位个位交换*/
    return m;
    }
